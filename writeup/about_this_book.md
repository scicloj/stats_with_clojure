# About This Book

Starts with Clojure is a book that's written for Clojure programmers who want to learn statistics. We in SciCloj community believe that Clojure is excellent for statistics and data science and hence this book exists to educate Clojure programmers about the mathematics of statistics.

One needs to know Clojure to read this book. But one may not need to know statistics as this book teaches statistics from the ground up.
