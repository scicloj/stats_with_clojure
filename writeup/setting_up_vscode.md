# Setting up Visual Studio Code

This book recommends using Visual Studio Code or its derivatives to use as the IDE to learn statistics with Clojure. One may download Visual Studio Code and install it following the instructions [here](https://code.visualstudio.com/download).

Once downloaded, Please install the following plugins:

- [Calva](https://marketplace.visualstudio.com/items?itemName=BetterThanTomorrow.calva)
- [Calva Power Tools](https://marketplace.visualstudio.com/items?itemName=BetterThanTomorrow.calva-power-tools)

Calva brings Clojure REPL to Visual Studio code. Calva Power Tools identifies you are opening a SciCloj project and jacks in your project.
